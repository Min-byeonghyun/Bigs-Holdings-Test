# Bigs Holdings Test

---

## 실행 방법

1. **프로젝트 클론**

   ```bash
   git clone https://github.com/Min-byeonghyun/Bigs-Holdings-Test.git
   cd Bigs-Holdings-Test
   ```

2. **패키지 설치**

   ```bash
   npm install
   ```

3. **개발 서버 실행**
   ```bash
   npm run dev
   ```

## ⚙️ 환경 변수(.env) 관련

- 별도의 .env 파일 없이 실행 가능합니다.
- API 주소는 하드코딩되어 있습니다.
